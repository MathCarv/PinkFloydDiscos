## Introdução

Esse projeto tem como tema a banda Pink Floyd, onde é possível ver diversas curiosidades ao longo do tempo sobre a banda e os discos. 

## Como utilizar?

### Clone o Repositório
```
git clone https://github.com/MathCarv/PinkFloydDiscos
```
### Entre no Diretório
```
cd PinkFloydDiscos
```
### Execute o comando
```
flutter run
```
